# Portfólio Pessoal — Projeto Integrador TDSV1

Site pessoal desenvolvido como **Projeto Integrador** das disciplinas de
**Introdução à Informática** e **Lógica de Programação I** do curso de
Tecnologia em Desenvolvimento de Sistemas.

## 📁 Estrutura de arquivos

```
meu-site/
├── index.html        → Página inicial com posts em destaque
├── projetos.html     → Galeria de projetos
├── curriculo.html    → Linha do tempo com formação e habilidades
├── blog.html         → Listagem completa de posts
├── contato.html      → Formulário de contato e especificações de hardware
└── assets/
    ├── css/
    │   └── estilo.css
    ├── js/
    │   └── app.js
    └── imagens/
        └── (foto-perfil.jpg, ícones...)
```

## 🛠 Tecnologias utilizadas

- HTML5 semântico
- CSS3 com variáveis CSS (custom properties)
- JavaScript puro (sem bibliotecas ou frameworks)
- Google Fonts
- GitHub Pages (hospedagem)
